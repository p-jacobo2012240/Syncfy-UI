Carga Programada Automática

¿Cuándo usar el proceso automático?

Útil para tareas de carga masivas, en donde se deben cargar muchos archivos hacia un bucket, en donde se tiene un path o varios paths de orígenes de archivos.

Carga Manual

¿Cuándo usar el proceso manual?

Útil para tareas cortas y lotes pequeños de archivos donde se desea cargar de forma manual los archivos, pero siempre con todas las opciones anteriores.
